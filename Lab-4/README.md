# Lab 4

Simulation, dynamics, and trajectory analysis tools for the later MECH 498 workflow.

## Contents

- `RobBase.py`: base simulation robot class.
- `RobStudent.py`: student implementation area.
- `RobSimulation.py`: simulation driver.
- `RRBot.py`: simpler robot model used in the lab.
- `rob_data.py`: data logging and plotting helpers.
- `rob_dynamics...so`: compiled dynamics support libraries.
- `robot_components.py`, `drawing_helper.py`, `general_utility.py`: shared support code.
- `test_rob.py`, `test_rr_bot.py`: local test scripts.
- `498_2026_lab4.pdf`: assignment handout.

## Goal

Work with motion simulation and supporting analysis tools beyond pure FK / IK.
