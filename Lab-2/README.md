# Lab 2

Forward-kinematics focused lab introducing the course robot framework.

## Contents

- `fanuc.py`: Fanuc robot model used in the lab.
- `robot_components.py`: reusable `Joint` / `Link` drawing and transform helpers.
- `drawing_helper.py`: 3D rendering support.
- `general_utility.py`: validation and helper functions.
- `test_script_student.py`: local test runner.
- `498-2026-lab2.pdf`: assignment handout.

## Goal

Implement and test robot forward kinematics while learning the transform and visualization pipeline used later in the course.
