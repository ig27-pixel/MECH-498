# Pitch Deck

RoboRoll Coatings investor pitch deck — 18 slides covering the live pitch,
leave-behind technical details, and transition slides.

## Files

- `roboroll_pitch_deck.html`: primary browser-presentable deck (scroll or use
  Page Down to advance slides).
- `roboroll_pitch_deck.pdf`: PDF export — print `roboroll_pitch_deck.html` from
  Chrome or Edge using **Print → Save as PDF** (backgrounds will render
  correctly due to the `print-color-adjust: exact` CSS rule).
- `roboroll_pitch_deck.pptx`: PowerPoint export generated by
  `export_pitch_deck.py`.
- `images/`: generated mockup PNG files used by the HTML and PPTX.

## Deck Structure

| Slides | Content |
| --- | --- |
| Title | Hero slide with robot render |
| 1 – 5 | Live pitch: problem, why it matters, solution, demo, why we win |
| Thank You | End of live pitch |
| Leave-Behind | Transition slide |
| A1 – A10 | Technical, market, and roadmap deep-dives |

## Notes

- Mockup images are generated renders, not real photographs.
- Slide A6 uses the real dynamics plots from `../section_4_dynamics/` rather
  than a generated mockup.
- `pitch_deck_plan.md` is listed in `.gitignore` and is not tracked by git.
