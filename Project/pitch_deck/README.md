# Pitch Deck

leave-behind technical details, and transition slides.

## Deck Structure

| Slides | Content |
| --- | --- |
| Title | Hero slide with robot render |
| 1 – 5 | Live pitch: problem, why it matters, solution, demo, why we win |
| Thank You | End of live pitch |
| Leave-Behind | Transition slide |
| A1 – A10 | Technical, market, and roadmap deep-dives |
