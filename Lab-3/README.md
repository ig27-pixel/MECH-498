# Lab 3

Inverse-kinematics and task-level motion work for the course robot stack.

## Contents

- `fanuc_provided.py`: provided Fanuc robot reference implementation.
- `fanuc_ik...so`: compiled IK helper library used by the lab.
- `picasso.py`: drawing / painting-style application built on IK.
- `robot_components.py`, `drawing_helper.py`, `general_utility.py`: shared support code.
- `basketball.yaml`, `prism.yaml`, `tetra.yaml`: target / scene data.
- `test_script_student.py`: local test runner.
- `498-2026-lab3.pdf`: assignment handout.

## Goal

Use IK to move the end effector to target poses and connect kinematics to a more application-like robot task.
